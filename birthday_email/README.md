# Birthday Email

A professional Python application part of a larger full-stack portfolio.

## 🚀 Technical Stack
- **Language:** Python 3.10+
- **Libraries:** pandas, smtplib, datetime

## 🏗️ Architecture & Data Flow
Monitors a CSV database for birthdays, selects a random template, and automates personalized email delivery via SMTP.

## 📡 External APIs & Databases
- SMTP (Simple Mail Transfer Protocol), Local CSV Database

## 🛠️ Challenges Overcome
- Handling CSV parsing with pandas and managing secure SMTP connections.

## 💻 Installation (WSL/Linux)
1. Ensure Python 3 is installed.
2. Install dependencies: `pip install pandas`
3. Run the application:
   ```bash
   python3 main.py
   ```
