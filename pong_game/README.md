# Pong Game

A classic Pong game built using Python's Turtle graphics.

## 🚀 Technical Stack
- **Language:** Python 3.10+
- **Libraries:** turtle, time

## 🏗️ Architecture & Data Flow
Real-time physics simulation using Turtle graphics, handling collision detection and automated AI movement.

## 📡 External APIs & Databases
- Turtle Graphics

## 🛠️ Challenges Overcome
- Implementing collision physics and smooth real-time animation loops.

## 💻 Installation (WSL/Linux)
1. Ensure Python 3 is installed.
2. Install dependencies: `sudo apt install python3-tk`
3. Run the application:
   ```bash
   python3 main.py
   ```
