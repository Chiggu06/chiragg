# Password Manager

A professional Python application part of a larger full-stack portfolio.

## 🚀 Technical Stack
- **Language:** Python 3.10+
- **Libraries:** tkinter, json, pyperclip

## 🏗️ Architecture & Data Flow
GUI-based tool for generating, searching, and storing credentials in a local JSON database.

## 📡 External APIs & Databases
- Local JSON Database

## 🛠️ Challenges Overcome
- Robust error handling for file I/O and managing complex Tkinter grid layouts.

## 💻 Installation (WSL/Linux)
1. Ensure Python 3 is installed.
2. Install dependencies: `sudo apt install python3-tk && pip install pyperclip`
3. Run the application:
   ```bash
   python3 main.py
   ```
