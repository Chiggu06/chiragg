# Snake Game

A professional Python application part of a larger full-stack portfolio.

## 🚀 Technical Stack
- **Language:** Python 3.10+
- **Libraries:** turtle, random, time

## 🏗️ Architecture & Data Flow
Classic arcade simulation with high-score persistence, collision detection, and coordinate-based growth.

## 📡 External APIs & Databases
- Local Text Database (data.txt), Turtle Graphics

## 🛠️ Challenges Overcome
- Implementing persistent high scores and managing segment-following logic.

## 💻 Installation (WSL/Linux)
1. Ensure Python 3 is installed.
2. Install dependencies: `sudo apt install python3-tk`
3. Run the application:
   ```bash
   python3 main.py
   ```
